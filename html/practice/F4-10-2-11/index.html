JavaScript按钮跳转功能

学习如何使用JavaScript实现按钮点击跳转到其他网站的各种方法，包含完整代码示例和视觉效果

基本页面跳转

使用window.location.href实现当前页面跳转到指定URL

// 获取按钮元素
const btn = document.getElementById('basicJumpBtn');

// 添加点击事件监听器
btn.addEventListener('click', function() {
  // 跳转到新页面
  window.location.href = 'https://example.com';
});

新标签页打开

使用window.open在新标签页中打开URL

const btn = document.getElementById('newTabBtn');

btn.addEventListener('click', function() {
  // 在新标签页打开URL
  window.open('https://example.com', '_blank');
});

延迟跳转

使用setTimeout实现延迟一段时间后跳转

const btn = document.getElementById('delayedJumpBtn');

btn.addEventListener('click', function() {
  // 显示倒计时提示
  alert('将在5秒后跳转到示例网站...');

  // 5秒后跳转
  setTimeout(function() {
    window.location.href = 'https://example.com';
  }, 5000); // 5000毫秒 = 5秒
});

确认后跳转

添加确认对话框，用户确认后再跳转

const btn = document.getElementById('confirmJumpBtn');

btn.addEventListener('click', function() {
  // 显示确认对话框
  const confirmed = confirm('您确定要跳转到示例网站吗？');

  // 如果用户点击"确定"
  if (confirmed) {
    window.location.href = 'https://example.com';
  }
});

动态URL跳转

根据用户输入或选择动态决定跳转目标

const btn = document.getElementById('dynamicJumpBtn');
const selector = document.getElementById('siteSelector');

btn.addEventListener('click', function() {
  // 获取选择的URL
  const targetUrl = selector.value;

  // 跳转到选择的网站
  window.location.href = targetUrl;
});

带参数的跳转

在URL中添加查询参数传递数据

const btn = document.getElementById('paramJumpBtn');

btn.addEventListener('click', function() {
  // 定义要传递的参数
  const userId = '12345';
  const source = 'demo-page';

  // 构建带参数的URL
  const url = `https://example.com/profile?user=${userId}&source=${source}`;

  // 跳转到带参数的URL
  window.location.href = url;
});
